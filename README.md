🐾 Banho & Tosa — Sistema de Gestão para Pet Shops
Sistema completo de gestão para pet shops desenvolvido com Flutter + Firebase.

🚀 Como Executar
Pré-requisitos

Flutter SDK instalado
Projeto Firebase configurado

Passos
bashgit clone https://github.com/seu-usuario/banho-tosa.git
cd banho-tosa
flutter pub get
flutter run -d chrome

✨ Funcionalidades

👥 Cadastro de clientes e pets
📅 Agendamento de serviços
💰 Controle financeiro e relatórios
📦 Controle de estoque com alertas
🔔 Notificações automáticas de agendamento
💬 Integração com WhatsApp
👤 Autenticação completa com foto de perfil


🛠️ Tecnologias

Flutter — Framework multiplataforma
Firebase Auth — Autenticação
Firestore — Banco de dados
